Portrait Seam Fixer — The Ivalice Chronicles
=============================================

Fixes the black horizontal line that appears on animated portrait
textures when converting or editing .dds / .png portrait files.

USAGE

  Drag and drop your portrait(s) onto fix_portrait_seam.py
  OR
  Open fix_portrait_seam.py and use the Browse buttons in the GUI.

FIRST TIME SETUP

  1. Install Python from https://python.org  (check "Add to PATH")
  2. Double-click fix_portrait_seam.py — it installs its own
     dependencies automatically on the first run.
